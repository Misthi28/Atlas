# ATLAS Destination Partner

**ATLAS Destination Partner** is a smart travel-dashboard web application designed to enhance travel experiences through intelligent insights.  
It provides weather updates, predicted tourist flow, event details, route suggestions, and real-time feedback sentiment â€” all in one place.

##  Live Demo  
 Website link : https://atlasdestinationpartner.netlify.app/

---

##  Features  
-  Destination-based interactive dashboard  
-  Real-time weather updates and forecasts  
-  Tourist flow prediction for major attractions  
-  Suggested itineraries and travel routes  
-  Local events and festivals listing  
-  Blogs and tourist reviews  
-  Sentiment analysis of visitor feedback  
-  Responsive design for mobile and desktop

---

## Tech Stack  
- **Frontend:** HTML5, CSS3, JavaScript  
- **Styling:** Tailwind CSS  
- **APIs:** Weather API, Events API, Custom Sentiment Analysis API  
- **Hosting:** Netlify  
- **Version Control:** Git & GitHub

## Installation  

To run this project locally:

```bash
# Clone the repository
git clone https://github.com/your-username/atlas-destination-partner.git

# Navigate into the directory
cd atlas-destination-partner

# Install dependencies
npm install

# Run the project
npm start
